Tentang

Assalamualaikum Warohmatullahi Wabarakatu....... Perkenalkan nama saya Sity Fatima Rumalean, biasa dipanggil Sity atau Fatima. Saya dari kelas 1A dengan NIM 16323001. Saya lahir di Fakfak, 11 Maret 2005. Sekarang umur saya 19 tahun, golongan darah AB, warna kesukaan biru, pink, dan putih. Hobi saya membaca. Lagu favorit Kenangan. Saya beragama Islam, asal daerah SBT Seram Bagian Timur. Saya anak pertama dari 3 bersaudara. Saya bersekolah di SD Inpres 1 Wagom, kemudian melanjutkan SMP di Madrasah Tsanawiyah Negeri Fakfak, dan SMA di Madrasah Aliyah Negeri Fakfak jurusan IPA. Setelah lulus, saya melanjutkan kuliah di Politeknik Negeri Fakfak melalui jalur SNPMB dan memilih dua jurusan yaitu D3 Manajemen Informatika dan D3 Teknik Sipil. Saya lolos di jurusan D3 Manajemen Informatika. Sekarang saya semester 2. Alamat tempat tinggal saya di Jalan Yos Sudarso RT 29. Saya sangat suka membaca Wattpad dan saat ada waktu luang biasanya saya gunakan untuk menonton TikTok. Makanan kesukaan saya yaitu nasi goreng, mie goreng, dan ayam goreng. Saya mempunyai dua adik laki-laki. Adik laki-laki saya yang pertama bersekolah di Madrasah Aliyah Negeri Fakfak dan adik laki-laki kedua saya bersekolah di SD Inpres 1 Wagom. Saat ini saya tinggal bersama kedua orang tua saya dan adik-adik saya. Kegiatan sehari-hari saya yaitu berkuliah dan setelah pulang kuliah, saya menghabiskan waktu luang untuk menonton TikTok dan membaca Wattpad di rumah. Saya memiliki 3 sahabat yang bernama April, Tasya, dan Aulia. Di kampus, saya juga memiliki 3 sahabat yaitu Hasri, Dian, dan Sri. Saya sangat suka memasak nasi goreng, walaupun menurut orang lain rasanya tidak enak, tetapi bagi saya sangat enak.
